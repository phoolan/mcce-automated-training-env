# Modul A – HTML Tabelle

## Ziel
In diesem Modul soll eine statische HTML-Seite erstellt werden, die eine Produktliste in Form einer Tabelle darstellt.  
Der Fokus liegt auf sauberem, semantischem HTML.

---

## Aufgabenstellung

Erstelle eine HTML-Seite `index.html` mit folgenden Anforderungen:

1. Die Seite enthält eine Überschrift mit dem Text **„Produktliste“**.
2. Darunter befindet sich eine HTML-Tabelle.
3. Die Tabelle besitzt folgende Spalten:
   - ID
   - Name
   - Preis (EUR)
   - Kategorie
4. Die Tabelle enthält **mindestens drei Datenzeilen** mit Beispiel-Daten.
5. Die Tabelle ist korrekt strukturiert und verwendet:
   - ein `<thead>`-Element für die Tabellenüberschriften
   - ein `<tbody>`-Element für die Tabelleninhalte

---

## Beispiel-Daten

Die folgenden Daten können verwendet werden:

| ID | Name     | Preis (EUR) | Kategorie  |
|----|----------|-------------|------------|
| 1  | Keyboard | 39.90       | Periphery  |
| 2  | Mouse    | 19.90       | Periphery  |
| 3  | Monitor  | 199.00      | Display    |

---

## Technische Vorgaben

- Es darf **nur HTML** verwendet werden.
- Es ist **kein JavaScript** und **kein Backend** erlaubt.
- Die Datei muss den Namen `index.html` tragen.

---

## Abgabe

- Abzugeben ist ausschließlich die Datei `index.html`.
- Die Datei muss im Root-Verzeichnis des Repositories liegen.
